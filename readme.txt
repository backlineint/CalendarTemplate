This archive contains snippets of code related to the calendar view available here:

http://mahindrahumanities.fas.harvard.edu/calendar-month?date=2014-04

The calendar is based on the default calendar view, but customized to match a desired design.

Files included:

* calendar_view_export.php - an export of the associated view.

* havard.css - an excerpt of the main css for the site - specifically focused on content related to the calendar.

* harvard.js - an excerpt of the main javascript for the site - specifically focused on content related to the calendar.

* page--calendar-month.tpl.php - page template for montly calendar
